Below here is a memory game in 3 different categories which are countries flags, food, and clothing.

The rules of the game are:

  • There is a 4x4 grid memory game, and each square will have an image of a country flag, food item or traditional clothing depending on the category you choose.
  • As soon as you start playing the memory game, the timer will automatically start, and it can be seen above the grid.
  • As soon as you make a move in the game, it will start counting the number of moves and that can be seen below the timer just above the grid.
  • If the picture matches each other then it will remain open, and a blue box will be highlighted around the square.
  • If the picture doesn't match the other picture, then it will automatically flip/close the image and no box will be highlighted.
  • When you complete the game then a message will pop up of you congratulating you to complete the game and it will show in how many moves and time you took to complete the game.
  • Have fun, enjoy the game and learn about culture
